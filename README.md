# dotfiles
My custom configurations for common dot files

## Install

```make install```

## Personal Cheat Sheet

| Shortcuts                            | Action                    |
|--------------------------------------|---------------------------|
| <kbd>CTRL</kbd>+<kbd>^</kbd>         | Previous buffer           |
| <kbd>CTRL</kbd>+<kbd>p</kbd>         | Fuzzy finder              |
| <kbd>g</kbd><kbd>d</kbd>             | Go to local definition    |
| <kbd>g</kbd><kbd>D</kbd>             | Go to global definition   |
| <kbd>c</kbd><kbd>g</kbd><kbd>n</kbd> | Change match under cursor |
| <kbd>c</kbd><kbd>i</kbd><kbd>w</kbd> | Change inner word         |
| <kbd>.</kbd>                         | Repeat previous command   |
